### ABOUT

Created for fun to test out a prototype app where the user only has a single button and has to set the time and an alarm.

Current progress:

- Loading / App Icon
- Intro
- Set am / pm, hour, and minutes with single tap
- Change 'focus' with double tap
- Hold to set Time
- Confirmation
